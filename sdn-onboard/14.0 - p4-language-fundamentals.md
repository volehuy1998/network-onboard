## Mục tiêu bài học

Sau khi hoàn thành phần này, bạn có thể:

1. **Mô tả (Understand)** cấu trúc P4_16 program: headers + parser + control blocks + tables + actions + deparser.
2. **Áp dụng (Apply)** viết P4 program cơ bản cho L2/L3 forwarding trên BMv2 simulator.
3. **Phân biệt (Analyze)** PSA (Portable Switch Architecture) externs so với target-specific externs (Tofino, BMv2).

*Skeleton — nội dung sẽ bao gồm:* PSA standard block sequence (parser → ingress control → deparser → packet_in/out), PSA externs (Counter, Meter, Register), PSA vs V1Model.

*Skeleton — nội dung sẽ bao gồm:* PISA execution model, match-action unit internals, stage resources (TCAM + SRAM + ALU).
