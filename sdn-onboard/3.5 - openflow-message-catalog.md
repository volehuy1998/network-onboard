> **Trạng thái skeleton:** Phase F v3.7-Reckoning, 2026-04-26. Content fill ở Phase G cohort C4 (cornerstone OFPT_* core) + cohort P6/P7/P8 (peripheral OFPT_* + OFPMP_* + bundle).

## Mục tiêu bài học

Sau Phần này, học viên có thể:

1. Liệt kê 16 OFPT_* message types + 14 OFPMP_* multipart sub-types từ memory.
2. Cho mỗi message: nêu rõ direction (controller→switch hay reverse), trigger condition, body fields chính, error scenario.
3. Quan sát message trên wire qua `tcpdump` Wireshark dissector + decode hex.
4. Debug message handshake/exchange failure qua tcp dump + ovs-ofctl monitor.

## Yêu cầu kiến thức

- Phần 3.1 (OF 1.0 spec foundation), 3.3 (protocol messages overview), 3.4 (version difference).
- Wireshark + OF dissector basic.
