## Dàn bài

## 13.1.1 Three-tier: NBDB → ovn-northd → SBDB → ovn-controller → OVS

*Sẽ phát triển: data flow architecture; north-south separation; translation pipeline; diagram từ CMS đến hypervisor OVS.*

## 13.1.2 NBDB tables overview

*Sẽ phát triển: Logical_Switch, Logical_Switch_Port, Logical_Router, Logical_Router_Port, ACL, Load_Balancer, NAT, Address_Set, Port_Group, Meter, DNS; quan hệ foreign keys.*

## 13.1.3 SBDB tables overview

*Sẽ phát triển: Logical_Flow, Multicast_Group, Meter, Port_Binding, MAC_Binding, Chassis, Encap, Datapath_Binding, IGMP_Group; physical state reflection.*

## 13.1.4 ovn-northd translator daemon

*Sẽ phát triển: monitor NBDB changes; compute logical flows; write to SBDB; performance optimization (incremental processing "northd incremental" from OVN 21.03); HA với active/standby.*

## 13.1.5 ovn-controller per-chassis registration

*Sẽ phát triển: runs on each hypervisor/gateway; register Chassis row in SBDB; receive Logical_Flow relevant to local ports; translate to OpenFlow rules on br-int.*
