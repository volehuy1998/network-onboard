## Dàn bài

## 15.1.1 OPNFV launch 2014

*Sẽ phát triển: Open Platform for NFV announcement September 2014 by Linux Foundation; founding members (AT&T, Brocade, Cisco, Dell, Ericsson, HP, Huawei, IBM, Intel, Juniper, NEC, Nokia, Red Hat, Telecom Italia, Vodafone); mission: carrier-grade NFV platform integration test.*

## 15.1.2 Release train: Arno → Gambia

*Sẽ phát triển: release timeline bắt đầu Arno (June 2015); Brahmaputra (2016), Colorado, Danube, Euphrates, Fraser, Gambia (2018); version highlights.*

## 15.1.3 Merger với CNTT → Anuket 2020

*Sẽ phát triển: CNTT (Common NFVi Telco Taskforce) at LFN; Anuket formed March 2021 by merger OPNFV + CNTT; RA (Reference Architecture) deliverables.*

## 15.1.4 Current state 2026

*Sẽ phát triển: status projects dưới LFN umbrella; Reference Implementation 2; operator adoption; relationship với ONAP (Open Network Automation Platform).*
