## Mục tiêu bài học

Sau khi hoàn thành phần này, bạn có thể:

1. **Mô tả (Understand)** P4Runtime gRPC API: ForwardingPipelineConfig (load/modify/get), WriteRequest (install/modify/delete table entries), ReadRequest (query counters/registers), StreamChannel (packet_in/out + errors).
2. **Phân biệt (Analyze)** P4Runtime (pipeline programmability) vs OpenFlow (fixed pipeline), hiểu tại sao P4Runtime schema-based vs OpenFlow spec-based.
3. **Áp dụng (Apply)** kết nối P4-capable switch (BMv2 hoặc Tofino) với ONOS controller qua P4Runtime, install một table entry runtime.
