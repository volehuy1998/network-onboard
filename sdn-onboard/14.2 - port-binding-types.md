## Dàn bài

## 14.2.1 Port_Binding overview

*Sẽ phát triển: SBDB Port_Binding table schema; columns (logical_port, datapath, chassis, tunnel_key, type, options); binding life cycle.*

## 14.2.2 vif binding — VM port

*Sẽ phát triển: default empty type; VM tap port attach via Nova/libvirt; chassis column set khi VM spawned; iface-id external_id matching.*

## 14.2.3 localnet — underlay provider

*Sẽ phát triển: localnet type; options: network_name mapping; VLAN tag; multi-chassis (global availability); use cases provider/flat/VLAN network.*

## 14.2.4 chassisredirect — gateway offload

*Sẽ phát triển: chassisredirect type; used cho distributed-gateway-port with HA_Chassis_Group; traffic redirect đến active chassis; SNAT location.*

## 14.2.5 patch — logical router cross-datapath

*Sẽ phát triển: patch type; peer logical port; represents L2↔L3 crossing (LS to LR); transparent to tunnel encap.*

## 14.2.6 localport — same-chassis metadata

*Sẽ phát triển: localport type; not routed across chassis; use cases: metadata service port, DHCP relay per-chassis; multiple chassis có cùng localport.*

## 14.2.7 l3gateway — distributed gateway port

*Sẽ phát triển: l3gateway type; gateway_chassis options; DGP (distributed gateway port) legacy naming; HA failover.*
