## Dàn bài

## 11.0.1 VXLAN RFC 7348 Aug 2014

*Sẽ phát triển: Cisco/VMware co-authored; UDP 4789 (IANA assigned); 24-bit VNI; outer IP/UDP + VXLAN header + inner Ethernet; flood-and-learn data plane.*

## 11.0.2 Geneve RFC 8926 Nov 2020 — GENEric Network Virtualization Encapsulation

*Sẽ phát triển: 8-byte fixed header + variable TLV options; IANA Option Class 0x0105 reserved OVN; UDP 6081.*

## 11.0.3 STT (Stateless Transport Tunneling) — Nicira draft, expired

*Sẽ phát triển: STT leverage TCP offload (TSO/LRO) for virtual; Internet-Draft expired 2014; why abandoned.*

## 11.0.4 GRE, MPLSoGRE — legacy

*Sẽ phát triển: GRE RFC 2784/2890; MPLSoGRE for service provider; không scale cho multi-tenant DC.*

## 11.0.5 Trade-off table: header size / TLV / hardware offload / control plane compat

*Sẽ phát triển: bảng so sánh 4 protocols; hardware offload support per NIC vendor; compatibility với existing L3 fabric.*
