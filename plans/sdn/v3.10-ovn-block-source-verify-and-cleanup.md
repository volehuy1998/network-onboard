### 3.3. Five verification primitives

#### Primitive A: function-name citation

```bash
cd /c/Users/voleh/Documents/ovn
git checkout v22.03.8 -q
grep -nE "^<fn>" <path>          # column-0 anchor per OVN coding style
```

If empty, fall back:

```bash
grep -rnE "^<fn>" controller/ northd/ lib/ include/ utilities/
```

If still empty: category = `FABRICATED`.

If non-empty and curriculum cites a specific line: compare. Mismatch = `WRONG_LINE`.

#### Primitive B: URL citation

```bash
# Inspect the URL pattern: https://github.com/ovn-org/ovn/blob/<ref>/<path>[#<anchor>]
# Acceptable <ref>: v22.03.8 | branch-22.03 | branch-24.03 | 40-char SHA
# Unacceptable <ref>: main | master | branch- without LTS suffix

# Verify the cited file resolves at the curriculum baseline
cd /c/Users/voleh/Documents/ovn
git checkout v22.03.8 -q
ls <path>                        # if missing: WRONG_PATH
```

If `<ref> = main`: category = `WRONG_BRANCH_URL`; decision = `FIX_BRANCH_URL` (rewrite to `branch-22.03`).

#### Primitive C: free-standing fact (pipeline stage ID, table number, opcode constant)

```bash
cd /c/Users/voleh/Documents/ovn
git checkout v22.03.8 -q

# Pipeline stage IDs are defined in northd/northd.c lines ~110-200 via the PIPELINE_STAGE macro
grep -nE "PIPELINE_STAGE\([A-Z]+, *(IN|OUT), *<STAGE_NAME_UPPER>" northd/northd.c
# Output format: PIPELINE_STAGE(SWITCH, IN, ARP_ND_RSP, 18, "ls_in_arp_rsp")
# Field 4 (index 3) is the table number.
```

If curriculum claim mismatches: category = `WRONG_FACT`; decision = `FIX_FACT`.

For OpenFlow opcode / register-bit constants in `controller/lflow.c`, `controller/lflow.h`, or `include/ovn/logical-fields.h`:

```bash
grep -nE "^#define <CONSTANT>" controller/ include/ northd/
```

#### Primitive D: schema table or column citation (Rule 14 §14.6)

```bash
cd /c/Users/voleh/Documents/ovn
git checkout v22.03.8 -q

# Step 1: list all tables (one-time per phase)
python3 -c "
import json
sb = json.load(open('ovn-sb.ovsschema'))
nb = json.load(open('ovn-nb.ovsschema'))
print('OVN-SB version', sb['version'], 'cksum', sb['cksum'], 'tables', len(sb['tables']))
print('OVN-NB version', nb['version'], 'cksum', nb['cksum'], 'tables', len(nb['tables']))
"
# Expected at v22.03.8:
#   OVN-SB version 20.21.0 cksum 2362446865 26963 tables 31
#   OVN-NB version 6.1.0   cksum 4010776751 31237 tables 27

# Step 2: confirm a specific table
python3 -c "
import json
d = json.load(open('ovn-sb.ovsschema'))
tname = '<TABLE>'
print(tname, 'EXISTS' if tname in d['tables'] else 'MISSING')
if tname in d['tables']:
    t = d['tables'][tname]
    print('  columns:', sorted(t['columns'].keys()))
    print('  isRoot :', t.get('isRoot'))
    print('  indexes:', t.get('indexes'))
"

# Step 3: confirm a specific column
python3 -c "
import json
d = json.load(open('ovn-sb.ovsschema'))
t = d['tables']['<TABLE>']
col = '<COLUMN>'
if col in t['columns']:
    print(col, 'EXISTS, spec:', t['columns'][col])
else:
    print(col, 'MISSING, available:', sorted(t['columns'].keys()))
"
```

If table missing: `FABRICATED_TABLE`. If column missing: `FABRICATED_COLUMN`. If `isRoot` mis-claimed: `WRONG_TYPE`.

#### Primitive E: commit SHA citation (Rule 14 §14.1)

```bash
cd /c/Users/voleh/Documents/ovn
git log -1 <short-sha-or-full-sha>        # expect: full SHA + author + date + message

# Cross-check inline 8-12-char SHA against References-section 40-char SHA
grep -nE "<short-sha>" <curriculum-file>
grep -nE "<full-sha>"  <curriculum-file>
```

If mismatch between inline and References: `SHA_INCONSISTENT`. If commit doesn't exist: `FABRICATED_SHA`.

- `URL-FUNCTION-PERMALINK` (URL with `#L<n>-L<m>` line anchor)
