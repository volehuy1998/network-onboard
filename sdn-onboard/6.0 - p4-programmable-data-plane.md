## Dàn bài

## 6.0.1 P4 origin: ACM CCR July 2014

Paper "P4: Programming Protocol-Independent Packet Processors". Bosshart, Daly, Gibb, Izzard, McKeown, Rexford, Shenker et al., ACM CCR 44(3), July 2014. Motivation: vượt OpenFlow limitations. OF spec định nghĩa cứng 12-tuple match + 8 action, không thêm được field mới mà không update spec. P4 cho phép mô tả parser + match-action pipeline + deparser fully programmable.

## 6.0.2 P4_14 vs P4_16 language evolution

P4_14 (2014-2016): verbose, tightly coupled với reference architecture. P4_16 (2016+): lean core language + PSA (Portable Switch Architecture) tách ra riêng. PSA định nghĩa standard blocks (parser, ingress, egress, deparser, packet_in/out) mà vendor implement. Từ 2019, P4_16 + PSA là standard.

## 6.0.3 PISA (Protocol-Independent Switch Architecture)

PISA là abstract machine model cho P4 target: programmable parser → match-action pipeline (N-stage) → programmable deparser. Mỗi stage có TCAM (ternary) + SRAM (exact) cho table lookup. Tofino là reference PISA silicon. Khác fixed-function ASIC (Broadcom Trident) ở chỗ pipeline là programmable, không phải hard-coded headers.

## 6.0.4 Targets: Tofino ASIC, BMv2 simulator, eBPF, DPDK

Tofino (Barefoot Networks 2016-2019, Intel 2019-2023): production silicon. BMv2 (behavioral model v2): software reference switch cho dev/test, không production-speed. T4P4S (DPDK target): P4 compile sang DPDK userspace. eBPF/XDP target: P4 → eBPF bytecode để chạy trên Linux. P4c compiler infrastructure hỗ trợ multiple backends.

## 6.0.5 P4Runtime API

gRPC-based southbound API cho controller program P4 device. Version-independent (một API cho nhiều P4 program versions). Standardized bởi ONF + ONOS collaboration 2018. Operation: install/delete/modify table entry, read counter/register, packet-in/packet-out. Thay thế OpenFlow cho P4-enabled device.

## 6.0.6 Intel acquisition 2019, Tofino EOL 2023, current state

Intel mua Barefoot $500M 2019. Plan: integrate Tofino vào Intel switch portfolio. 2023: Intel halt Tofino sales, impact: P4 ecosystem mất production target duy nhất. Current state 2026: P4 chủ yếu trong academic research, SONiC-DASH trial, và software targets (DPDK, eBPF). Tương lai commercial P4 vẫn mở. AMD Pensando DPU, Nvidia BlueField potential candidates.
