## Dàn bài

## 13.3.1 ACL: priority, direction, match, action

*Sẽ phát triển: ACL columns (direction=from-lport/to-lport; match expression; action allow/allow-related/drop/reject); priority range 0-32767; stateful via ct().*

## 13.3.2 Port_Group — reusable port collection

*Sẽ phát triển: Port_Group NBDB table; name-based reference trong ACL match; dynamic membership; Neutron security group backing.*

## 13.3.3 Address_Set — IP/MAC set

*Sẽ phát triển: Address_Set cho IP prefix / MAC sets; named reference in match; synced to SBDB Address_Set; CIDR syntax.*

## 13.3.4 Load_Balancer: VIP + backend pool

*Sẽ phát triển: NBDB Load_Balancer table; vips column map; health_check integration; attach to Logical_Switch or Logical_Router; protocols supported.*

## 13.3.5 NAT: DNAT/SNAT/DNAT_and_SNAT

*Sẽ phát triển: Logical_Router NAT rules; type enum (dnat, snat, dnat_and_snat); distributed vs centralized; floating IP pattern.*
