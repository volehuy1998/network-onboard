## Dàn bài

## 5.2.1 OCP Network project (2013+)

Facebook Open Compute Project khởi động 2011 cho server hardware disaggregation. OCP Network subproject launch 2013 mở rộng sang switching — mục tiêu: bare-metal switch với ONIE boot + NOS lựa chọn tự do. Specification open-source cho ODM thiết kế thiết bị tiêu chuẩn.

## 5.2.2 Whitebox hardware (Celestica, Edgecore, Delta)

ODM (Original Design Manufacturer) như Celestica, Edgecore, Delta sản xuất bare-metal switch theo OCP spec, bán giá thấp hơn OEM truyền thống 40-60%. Không preinstall NOS — khách tự cài. Hyperscale (Facebook, Microsoft, Google) mua từ ODM để giảm cost per port.

## 5.2.3 Merchant silicon: Broadcom Tomahawk, Trident

Broadcom thống lĩnh merchant silicon. Tomahawk series (TH1 2014, TH2 2016, TH3 2018, TH4 2020, TH5 2023) cho high-radix spine (25.6 Tbps throughput TH5). Trident series (Trident2 2013 → Trident5 2023) cho leaf. Các vendor khác: Marvell Prestera, Cisco Silicon One, Intel Tofino (EOL 2023).

## 5.2.4 NOS options: Cumulus Linux, SONiC, OpenSwitch, Stratum

Cumulus Linux: Debian-based NOS, acquired by Nvidia 2020, dùng Linux kernel cho forwarding. SONiC (Microsoft 2016, now Linux Foundation): container-based, SAI (Switch Abstraction Interface) abstracts silicon. OpenSwitch (HP 2015, abandoned 2017). Stratum (ONF P4-based runtime): thin switch agent, all intelligence trong controller.

## 5.2.5 ONIE (Open Network Install Environment)

Boot loader standard cho whitebox. Chạy initial boot, download + install NOS từ network (HTTP/HTTPS/TFTP). Tương tự PXE cho server. Interoperable — switch vendor-independent. ONL (Open Network Linux) là reference Linux để test ONIE boot flow.

## 5.2.6 Current state 2026: SONiC thắng thị phần hyperscale

Microsoft Azure, AliCloud, LinkedIn deploy SONiC production. SAI standardize silicon abstraction — vendor Broadcom/Nvidia/Marvell đều có SAI driver. Dự báo 2026: SONiC chiếm >50% hyperscale DC, Cumulus hold enterprise mid-market. Stratum niche trong research/P4 deployment.
