## 2. Phương pháp (4 phase tuần tự)

### Phase 1 — True gap audit (1 session)

**Vấn đề:** 165 keyword "MISSING" qua strict grep có nhiều **false-positive alias** (REF dùng tên dài, curriculum dùng tên ngắn).

**Action:**
- Refine `scripts/refine_coverage_matrix.py` thêm alias dictionary expansion (verb form, plural, abbreviation)
- Output `memory/sdn/keyword-true-gap-final.md` chỉ chứa **TRUE gap** (concept thực sự không có anywhere)
- Manual review top 50 entries để classify: REAL gap / alias miss / acceptable as-is

**Deliverable:** Ranked list ~30-80 TRUE gap (từ 165 strict MISSING).

**Acceptance:** Matrix refined, false-positive < 10% trong list TRUE gap.

---

### Phase 2 — TRUE gap fill (3-5 session)

**Action:** Mỗi TRUE gap = 1 commit nhỏ với 5-axis Anatomy thêm vào file native.

**Pattern:**
```
- Identify gap keyword X
- Identify file native dạy concept liên quan (qua master index 0.3 cross-link)
- Add 5-axis Anatomy section (Bucket | Context | Purpose | Activity | Mechanism)
- Add cross-link upstream source (Rule 14)
- Add Anatomy + 1 GE if applicable
- Update tracker
```

**Deliverable:** ~30-50 dedicated 5-axis Anatomy distributed across ~10-15 file existing.

**Acceptance:** Tier A MISSING ≤ 50 (giảm từ 165 xuống ~80% gap đã đóng).

---

### Phase 3 — SHALLOW upgrade (2-3 session)

**Action:** 56 keyword SHALLOW (mention 1-2 file nhưng chưa có 5-axis dedicated) → upgrade tới BREADTH+ qua focused expand.

**Pattern:**
```
- Pick SHALLOW keyword
- Find existing mention file (qua coverage matrix)
- Upgrade mention thành 5-axis Anatomy + Anatomy block
- Cross-link tới Phần liên quan
```

**Deliverable:** ~40 SHALLOW → BREADTH+ trong existing file.

**Acceptance:** Tier B SHALLOW ≤ 30 (giảm từ 56).

### Phase 4 — Validate + Release v3.6 (1 session)

**Action:**
- Re-run J.7-style audit cuối cùng
- Verify well-covered ≥ 250 (65% trên 383 in-scope)
- Update plan tracker, README, dependency-map (Rule 2)

**Deliverable:** v3.6 RELEASED.

**Acceptance:** Coverage matrix verified, all quality gate pass, tag + release published.
