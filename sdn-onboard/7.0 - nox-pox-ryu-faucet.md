## Dàn bài

## 7.0.1 NOX: Ethane's runtime, 2008

NOX opensource 2008 bởi Nicira — runtime cho Ethane (paper 2007) trở thành controller phổ biến đầu tiên. C++ event-driven architecture: ứng dụng register handler cho events (`packet_in`, `port_status`), controller dispatch khi nhận OpenFlow message. Khởi đầu: hỗ trợ chỉ OpenFlow 1.0 với 12-tuple match.

## 7.0.2 POX: Python port, 2011

POX là Python rewrite của NOX với mục đích teaching. Không nhắm production — performance chậm hơn NOX 10× do GIL + interpreted execution. Strength: Mininet integration tốt, syntax đơn giản cho lab exercise, tutorial abundant. Vẫn được dùng trong nhiều đại học CS giảng dạy SDN concept.

## 7.0.3 Ryu: NTT 2012, modular

NTT phát triển Ryu framework 2012, Python modular architecture. Hỗ trợ OpenFlow 1.0 đến 1.5 đầy đủ (có OF 1.5.1). Feature đặc biệt: BGP speaker built-in để controller vừa nói BGP vừa nói OpenFlow. Thư viện NX match extension tương thích với OVS NXM fields. Popular trong SDN research community.

## 7.0.4 Faucet: production focus, 2015+

Faucet fork Ryu với mục tiêu production (Melbourne Institute of Technology + REANNZ NZ collaboration). Thêm YAML config model (thay vì Python code), Gauge sidecar monitoring exporter, multi-table pipeline canonical cho campus/enterprise. Production: REANNZ (New Zealand academic network), một số ISP. Release cadence monthly. Mục đích khác Ryu: Faucet focus operability, Ryu focus flexibility.
